Kalkulator Sederhana


Pendahuluan

Kalkulator adalah alat bantu hitung elektronik yang biasanya digunakan untuk melakukan perhitungan, mulai dari aritmatika dasar hingga matematika kompleks. Kalkulator yang kita buat adalah kalkulator yang sederhana dan terbatas, kalkulator kami hanya dapat menghitung penjumlahan, pengurangan, perkalian, pembagian, dan perpangkatan. Kalkulator dibuat menggunakan koding dasar yang di buat sendiri oleh kami dan dibuat sesederhana mungkin agar mudah untuk dipakai dan mudah untuk dimengerti. Fungsi dari program kami adalah membantu seseorang untuk menghitung atau mereka yang mengalamai kesulitan dalam menghitung,kalkulator kami dapat membantu.


Fitur Utama :
- Penjumlahan
- Pengurangan
- Perkalian
- Pembagian
- Perpangkatan

Memang fitur kami tidak selengkap kalkulator lain karena ini hanya kalkulator sederhana. Namun ini sudah cukup untuk bisa menghitung matematika sederhana yang tidak terlalu rumit.

Panduan Instalasi

- Salin link repository kami
- Buka Aplikasi Visual Studio Code
- Buka Terminal
- Ketik “git clone <link-repo-kami>” atau bisa langsung "git clone https://github.com/Shannmengko/Kelompok2MLS_TugasVCS.git"
- Enter 

Panduan Menjalankan

- Jalankan/Run folder main.py
- Silahkan menghitung dengan memasukkan pilihan operasi matematika dan kedua angka yang akan digunakan
